## Overview

SCP-1048 (Builder Bear) is a premium custom role developed for SCP: Secret Laboratory communities and multiplayer servers.

The project expands the base game with a fully playable SCP entity featuring unique gameplay mechanics, custom animations, synchronized multiplayer interactions, and a dedicated schematic designed specifically for the role.

The primary goal was to create a role that feels visually distinctive while remaining seamlessly integrated into existing SCP:SL gameplay.
